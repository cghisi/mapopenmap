Canadian Tire Store Locator

Technology used :

  • OpenStreetMap: Collaborative Map of the World.
  • LeafletMap : javascript library to add markers, calculate distance ...
  • HTML5 Geolocation : Browser Geolocation API is used to locate a user's position.
  • IP Geolocation (extreme-ip-lookup.com) : Free IP Lookup Geolocation API. (limitation : 20 request/min - 50,000 requests per month)
